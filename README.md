# <a name="el"></a>GrabΑΔΑ
Μια εφαρμογή λήψης  αναρτηθέντων εγγράφων από το [diavgeia.gov.gr](http://diavgeia.gov.gr), με βάση τον ΑΔΑ.  
Copyright (c) 2014, [Πέτρος Κυλαδίτης](http://www.multipetros.gr)


Περιγραφή
=========
Μια εφαρμογή λήψης  αναρτηθέντων εγγράφων από το diavgeia.gov.gr, με βάση τον ΑΔΑ. 

Το __κατέβασμα__ της εκάστοτε ΔΑ, σε μορφή pdf, γίνεται δίνοντας στο σχετικό πλαίσιο που υπάρχει στο κεντρικό παράθυρο της εφαρμογής, τον ΑΔΑ. Κατόπιν με το πάτημα του κουμπιού _"Λήψη"_, πραγματοποιείται η λήψη του
εγγράφου.   

Το έγγραφο __αποθηκεύεται__ από προεπιλογή στον φάκελο εντός του οποίου βρίσκεται η εφαρμογή. 

Η __αλλαγή του φακέλου__ είναι δυνατή από το μενού
_"Ρυθμίσεις" -> "Επιλογή φακέλου αποθήκευσης"_. Επιστροφή στην προεπιλογή γίνεται από το μενού _"Ρυθμίσεις" -> "Αποθήκευση τον φάκελο του προγράμματος"_.

Στον φάκελο στον οποίο επιλέγετε να γίνεται η αποθήκευση θα πρέπει να έχετε __δικαιώματα εγγραφής__.

Τέλος, το __όνομα των αρχείων__ που προκύπτουν είναι της
μορφής: `ΑΡΙΘΜΟΣ-ΑΔΑ.PDF`

## Απαιτήσεις
Για τη λειτουργία της εφαρμογής απαιτείται υπολογιστής με MS Windows με εγκατεστημένο το __.NET 2.0__ framework, και ενεργή σύνδεση με το διαδίτυο.

Η εφαρμογή έρχεται σε portable μορφή, δίχως να απαιτείται οποιαδήποτε εγκατάσταση.

> Η επιλογή της συγκεκριμένης έκδοσης του .NET framework πραγματοποιήθηκε λόγω της υποστήριξής, αλλά και της παρουσίας του σε πολύ παλιά συστήματα Windows (XP, αλλά και παλαιότερα), καθώς τέτοια συναντούνται συχνά σε δημόσιες υπηρεσίες.

# <a name="en"></a>GrabADA
One-click downloader tool for greek governing documents from the project Diavgeia - [diavgeia.gov.gr](http://diavgeia.gov.gr)  
Copyright (c) 2014, [Petros Kyladitis](http://www.multipetros.gr)

## Description
GrabADA is an one-click downloader tool for greek  overning documents from the project Diavgeia by using the ADA (ΑΔΑ): Internet Post Number of the document.

The documents are downloaded at pdf format, in the folder of grabada’s executable, or at any other selected from the program menu.

## Requirements
It’s a portable application, no setup is necessary, just unzip and run the executable.

A computer with MS Windows and an active internet conection is required. Also the .NET 2.0 framework must be installed.

> The choice of the particular version of .NET framework took place because of its support, and also the presence, of a very old Windows systems (XP, and older), such as frequently encountered in the greek public sector.
